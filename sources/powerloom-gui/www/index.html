PowerLoom GUI Launch Page

NOTE: This page is best be viewed with Internet Explorer or Netscape Navigator 4.x.

The first time you execute the GUI, you will need to connect to a PowerLoom server. Obtain the hostname and port of a server from your system administrator, then select the KB -> Connect To Server menu item and enter the appropriate values. If you don't know the name of a server, try hostname = blackcat.isi.edu, port = 8000.

You may also browse the User's Guide, or peruse the JavaDocs for the GUI.
